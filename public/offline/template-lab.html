Visual Tuning

Whole Label Calibration

Scale and offset are preview/test calibration values. They shift and scale boxes, field origins, QR, logo, and simple font/field-block commands for Template Lab only. They do not change production printing until a template/profile is deliberately deployed.

QR

QR data always remains the lot number only. X/Y tune the native ^BQN origin; magnification controls printed module size.

Logo

Logo controls apply to the static ^GFA logo where enabled. If a profile has no logo, these controls are disabled. Use physical proof prints to judge final logo quality.

Field Fit

H = font height dots, W = font width dots. Large/medium/small/min are selected by text length; box width controls ^FB; no wrap/no hyphenation is enforced for constrained fields.

Product Description & Field Origins

Field X/Y overrides move the nearest fitted-field origin before that placeholder. Use product description X/Y to pull long descriptions back toward the left edge of their box.

Test Print / Export
